# Lagro Reviewer (React scaffold)

Development scaffold for the React + Firebase conversion.

Quick start:

1. Install dependencies

```bash
cd react-app
npm install
```

2. Start dev server

```bash
npm run dev
```

Place your Firebase config and API keys into `react-app/src/config.local.js` or expose them via `window.LAGRO_CONFIG` as in the root `config.js`.
